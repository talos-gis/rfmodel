RFModel
==========

The RFModel library is a frontend API for RF Models.

Currently the only supported model is:

* `TIREM`_ by `ALION`_ - see `tirem_python_api`_

Building `RFModel` wheel
~~~~~~~~~~~~~~~~~~~~~~~~~~~

  > build-rfmodel-wheel.bat
